# Bondi Skin Clinic

A premium, high-performance web application for **Bondi Skin Clinic**, a premier clinical cosmeceutical clinic located in Bondi, Australia. 

## 🌟 Overview
This website is designed to provide a luxury, clinical experience for clients seeking advanced skin, body, laser, and injectable treatments. It features a modern, responsive interface built with React and optimized for local SEO in the Bondi area.

## 🚀 Key Features
- **Local SEO Optimized**: Dynamic metadata (titles, descriptions, keywords) targeting the Bondi, Australia market using `react-helmet-async`.
- **Clinical Menu**: Comprehensive treatment catalog for Skin, Body, Laser, Injectables, and PRP.
- **Premium Aesthetics**: Sophisticated design using a custom color palette (Sand, Gold, Slate) and premium typography (Cormorant Garamond & DM Sans).
- **Dynamic Animations**: Smooth page transitions and interactive elements powered by `framer-motion`.
- **E-commerce Ready**: Integrated shop and cart functionality for clinical products.
- **Performance Focused**: Optimized asset loading, code splitting, and high-priority image fetching for fast LCP.
- **Infrastructure**: Automated `sitemap.xml` and `robots.txt` generation for search engine crawling.

## 🛠️ Technology Stack
- **Frontend**: React (Vite)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **SEO**: React Helmet Async
- **Icons**: Lucide React
